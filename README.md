# NewCsTeamServer

NewCsTeamServer 是一个用 Go 语言重写的 Cobalt Strike 服务端项目，目前已实现以下功能：

- **Agent 上线**：支持客户端（Agent）与服务端的通信。
- **命令下发**：服务端向客户端下发任务。
- **命令结果回传**：客户端执行任务后将结果回传至服务端。
